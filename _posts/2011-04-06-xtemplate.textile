h2. XTemplate 3.0

A linguagem XTemplate 3.0 é uma solução criada para facilitar a autoria de programas utilizando a linguagem NCL. Com XTemplate é possível criar templates, chamados de templates de composição, que descrevem de uma forma genérica todo o conteúdo de um contexto NCL. Ao programador cabe apenas indicar os elementos que deseja utilizar. É como se criássemos um documento NCL com "buracos" que poderíamos preencher posteriormente da forma como quisermos.

Para que um documento NCL usando templates possa ser executado em uma implementação padrão do middleware Ginga, é necessário o processamento deste documento transformando-o em um documento completo.

Para maiores informações sobre a linguagem, o processador e outras ferramentas relacionadas, a documentação do projeto está disponível na área de *"publicações":../../*. Um tutorial básico de uso da linguagem pode ser encontrado "(tutorial)aqui":../../tutoriais/xtemplate.
