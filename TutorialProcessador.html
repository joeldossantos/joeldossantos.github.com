Tutorial do processador de templates

 

O processador de templates possui, assim como apresentado anteriormente, uma interface que auxilia o usuário no processamento de um template.


Esta tela apresenta os botões selecionar, utilizado para selecionar um documento utilizando templates, e processar, utilizado para dar início ao processamento do documento selecionado. Além disso, a interface do processador também apresenta uma área de mensagens, onde o processador indica o andamento do processamento.


Quando o botão selecionar é pressionado, uma tela de escolha de arquivo é apresentada.


Assim que o documento utilizando templates é selecionado, é apresentada uma mensagem, na área de mensagens, informando o documento selecionado. O usuário pode então pressionar o botão processar para dar início ao processamento do documento utilizando templates.


Durante o processamento do documento utilizando templates, o processador apresenta mensagens informando o andamento do processamento, além de informações sobre os elementos processados. Quando o processamento chega ao fim, um popup é apresentado indicando se o processamento foi realizado com sucesso ou não.
